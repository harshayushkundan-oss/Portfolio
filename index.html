03 / Projects

Featured Work

PROJECT 01
Loan Funnel Analysis
SQL Power BI DAX
Problem Statement
The lending team had no visibility into where applicants were dropping off across the loan funnel — from application to disbursement — making it impossible to optimize conversion at each stage.
Approach
Extracted and modeled multi-stage application data using SQL. Built DAX measures to calculate stage-wise conversion rates and drop-off percentages. Designed an interactive Power BI funnel dashboard with drill-down by region, product type, and time period.
Key Insights
Identified that 42% of drop-offs occurred at the income verification stage. Peak abandonment correlated with application times after 6 PM, suggesting a UX friction point in the verification workflow.
Reduced overall funnel drop-off by 15% through targeted operational changes informed by the dashboard insights.
PROJECT 02
Sales Performance Dashboard
Power BI Excel SQL
Problem Statement
Leadership had no single source of truth for sales performance across 100,000+ records. Reports were manually compiled in Excel, taking hours each week and often containing errors or outdated figures.
Approach
Cleaned and modeled 100,000+ sales records in SQL and Excel. Built a fully automated Power BI dashboard with real-time refresh, covering revenue trends, YoY growth, top-performing SKUs, and regional breakdowns with dynamic slicers.
Key Insights
Surfaced that 3 product categories accounted for 68% of revenue, while 2 underperforming regions were consuming 30% of resources with minimal return — previously invisible in the old reporting process.
Eliminated 8+ hours of weekly manual reporting. Leadership reallocated resources from underperforming regions, recovering 12% revenue efficiency.
PROJECT 03
Customer Segmentation
Python Pandas Scikit-learn Matplotlib
Problem Statement
The marketing team was sending identical campaigns to all customers, resulting in low engagement and high unsubscribe rates. No behavioral segmentation existed to personalize outreach.
Approach
Applied RFM (Recency, Frequency, Monetary) analysis using Python and Pandas on customer transaction data. Used K-Means clustering to segment customers into 5 distinct behavioral groups. Visualized segment profiles with Matplotlib and Seaborn.
Key Insights
Identified a high-value "Champions" segment (8% of customers contributing 52% of revenue) and a large "At-Risk" segment that hadn't purchased in 60+ days — both previously treated identically in campaigns.
Targeted re-engagement campaign for "At-Risk" cluster achieved 34% reactivation rate vs. 9% baseline from generic campaigns.
